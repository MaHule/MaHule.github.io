02.字符串切片操作

java,python = "java","python"
jl,pl = len(java),len(python)

[str(java[i%3*jl:] + python[i%5*pl:] or i) for i in range(1,10)]

03.join连接字符串

mystr = ['I','love','Python']

res = '_'.join(mystr)
print(res)

I_love_Python

04.分割字符串

根据指定字符或字符串，分割一个字符串时，使用方法split（join和split可以看作做一对互逆操作）

res.split('_')

['I', 'love', 'Python']

05.替换

字符串替换使用replace方法

s = 'i love python'.replace('o','O')
s

'i lOve pythOn'

06.子串判断

判断a字符串是否为b字符串的字串

  • 方法1：使用in
a = 'our'
b = 'flourish'

r = True if a in b else False
r

True
  • 方法2：使用方法find，返回字符串b中匹配字串a的最小索引：
a = 'our'
b = 'flourish'

b.find(a)

2

07.去空格

清洗字符串时，位于字符串开始和结尾的空格，有时需要去掉，strip方法能实现

a = '  \tI love python  \b\n'
print(a)
print(a.strip())

I love python  

08.字符串的字节长度

encode方法对字符串编码后：

def str_byte_len(mystr):
    mystr_bytes = mystr.encode('utf-8')
    return (len(mystr_bytes))
